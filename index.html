# 常用命令

hugo new site <site-name>
# 启动本地服务
hugo server
hugo server -D

hugo
# 新文章
hugo new posts/first-post.md


# GitHubPages托管，github.io设置
1、github 创建仓库，名称： <yourname>.github.io
2、hugo.toml 设置 baseURL 为 https://<yourname>.github.io
3、hugo命令构建静态资源
4、上传构建成果到仓库中
5、浏览器打开 https://<yourname>.github.io
